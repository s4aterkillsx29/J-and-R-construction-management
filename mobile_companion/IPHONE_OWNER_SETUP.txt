J and R Construction Manager — iPhone Owner/Admin Quick Access

Use this when Jacob wants one-tap iPhone access to the owner/admin mobile dashboard.

QUICK PATH (verified flow)
1. PC: Start Center → Start Local Host → wait for verify.
2. iPhone Safari (NOT DuckDuckGo): open /connect on the same Wi-Fi.
3. iPhone Safari: /login with owner account.
4. iPhone Safari: Share → Add to Home Screen → "J&R Manager".
5. Open the home screen icon — launches /mobile (owner/admin mobile dashboard).

DUCKDUCKGO / CHROME ON IPHONE
- DuckDuckGo often shows "cannot access that page" for http://192.168.x.x addresses.
- Fix: copy the URL from Start Center → Mobile Links, open Safari, paste in the address bar.
- Do not use 127.0.0.1 or localhost on the phone — that is the phone, not your PC.

SETUP GUIDE URL (no login required)
http://YOUR-PC-LAN-IP:8765/mobile/setup

BOOKMARK URL (after login)
http://YOUR-PC-LAN-IP:8765/mobile

WHAT YOU GET ON PHONE (owner/admin)
- Mobile dashboard, jobs, pipeline board
- Job details with expenses and worker pay (when allowed)
- Mobile files / receipt search
- Full admin pages: payroll, bookkeeping, expenses (same login)

AWAY FROM SHOP
- Deploy cloud_hosting (Render/Railway/Fly/tunnel)
- Save Cloud Access URL in PC Start Center
- Phone uses cloud URL instead of LAN IP for bookmark

DO NOT
- Share admin password
- Use default passwords on a shared host
- Expect phone to work if PC host is off and no cloud URL is saved

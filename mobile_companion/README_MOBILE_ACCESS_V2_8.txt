J and R Construction Manager v2.8 Mobile Access

This is a mobile web app/PWA approach. It is the best immediate mobile access because users do not need App Store or Google Play installs.

How it works:
- One PC/cloud host runs the shared session server.
- Phones connect to the host URL through a browser.
- Users log in with their assigned role.
- Admins/managers can upload evidence if allowed.
- Workers/viewers have read-only job and file access.

Phone install:
- iPhone: Safari > Share > Add to Home Screen.
- Android: Chrome > menu > Add to Home screen.

Security:
- Use LAN/VPN for testing.
- Use HTTPS and firewall rules for cloud/public hosting.
- Never keep admin/admin for shared access.

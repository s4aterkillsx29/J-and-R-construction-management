J and R Construction Manager - Mobile Companion

This package includes a browser-based mobile companion/PWA mode.

How to use:
1. Start the shared host server on Jacob's admin laptop or cloud server.
2. From a phone on the same trusted network/VPN, open the host URL and log in.
3. Open /mobile or use the Mobile tab.
4. Use your phone browser menu to Add to Home Screen.

This is not a separate App Store app yet. It is the safest first mobile version because it uses the same secured host, users, roles, sessions, files, and database.

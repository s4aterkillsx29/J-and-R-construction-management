J & R Construction Manager v6.4 - Extreme Final Live Setup Gate Edition

Best install/update order:
1. Extract the ZIP.
2. Run !!! START INSTALL HERE.vbs.
3. Optional but recommended if an older copy is installed: click Pre-Install Login Check. This opens the existing secure app login/status without the installer storing any passwords.
4. Click Install / Update.
5. The installer preserves business data and opens First Setup/Login after install.
6. Log in inside the app, not the installer.
7. Change default admin/admin immediately if it is still active.
8. Run Self Setup + Verify and Extreme Final Verify.

Security rule: admin/admin is local first setup only and is blocked for customer/remote/cloud use once setup is complete or when access is not local.

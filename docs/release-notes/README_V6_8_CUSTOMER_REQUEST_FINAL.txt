JRC Construction Manager v6.8 - Customer Request Final Live Edition

Purpose:
This release finalizes the customer portal and customer job request workflow. It is designed so customer accounts can submit complete job requests, track only their own customer-facing information, and stay separated from J&R internal business records.

Customer request intake now documents:
- Customer/contact name
- Phone and/or email
- Preferred contact method
- Full property address
- Property type
- Occupancy status
- Tenant/site contact info when needed
- Request title
- Service type
- Priority
- Request reason
- Description of work needed
- Requested timeline
- Preferred appointment window
- Access notes
- Safety/hazard notes
- Pets/occupant notes
- Photo/attachment notes
- Optional budget/approval notes
- Required acknowledgement that the request is not an approved job, schedule, or final price

Customer safety:
Customers cannot see payroll, job costs, expenses, bookkeeping, internal notes, admin tools, hosting tools, device tools, full file explorer, or other customers.

Owner/admin workflow:
Owner/admin can review customer requests, update status, add customer-visible notes, and keep internal J&R notes hidden from customers.

After install:
1. Open Quick Setup Login.
2. Log in.
3. Run System Check.
4. Run Customer Request Final Check.
5. Test a customer account by creating a customer user and submitting one request.

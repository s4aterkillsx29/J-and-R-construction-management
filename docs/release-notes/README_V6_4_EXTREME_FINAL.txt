JRC Construction Manager v6.4 - Extreme Final Live Setup Gate Edition

Purpose:
- Adds Pre-Install Login Check before updates when an installed copy exists.
- Keeps installer from collecting/storing passwords.
- Preserves changed owner/admin password in the business database across installs.
- Keeps admin/admin as local first-setup only and blocked for customer/remote/cloud use.
- Adds Extreme Final Verify for install, login, roles, customer/external dashboards, cloud, Dropbox/file-source, and security checks.

Best order:
1. Run !!! START INSTALL HERE.vbs.
2. If updating an installed copy, click Pre-Install Login Check.
3. Log in in the existing app if it opens and confirm owner/admin status.
4. Click Install / Update.
5. Log in inside the secured app after install.
6. Run Self Setup + Verify and Extreme Final Verify.

Before sharing with customers/workers:

[ ] Change default admin/admin password if still present.
[ ] Run Self Setup + Verify.
[ ] Run System Check.
[ ] Run Permission View Check.
[ ] Run Security Perspective Audit.
[ ] Run Dashboard Role Check.
[ ] Run Live Deployment Readiness.

[ ] Set JRC_TRUSTED_HOSTS to your real domain/host.
[ ] Set JRC_CLOUD_BASE_URL to the real HTTPS URL.
[ ] Confirm /api/health, /api/connection, /connect, /login, /mobile, /register, /apply, /customer/request.
[ ] Create admin backup before inviting users.
[ ] Create test accounts for manager, worker, viewer, customer, and non-company.
[ ] Confirm each test account only sees its allowed dashboard.
[ ] Share only the correct live links.
